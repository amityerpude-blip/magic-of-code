🐼 What is Pandas?

Pandas is a powerful Python library used to organize, analyze and manipulate data easily.

📦 Series

A Series is a one-dimensional labelled data structure capable of storing any data type.

🗂 DataFrame

A DataFrame is a two-dimensional table consisting of rows and columns, similar to a spreadsheet.

📄 Reading Data

Use read_csv() to load CSV files into a DataFrame for analysis.

🔍 Exploring Data

Functions like head(), tail(), info() and describe() help us understand the dataset.
