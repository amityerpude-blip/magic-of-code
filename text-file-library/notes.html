📜 What is a Text File?

A text file is a collection of characters stored permanently on secondary storage. Python can create, read, modify and delete text files.

✨ Opening a File

file = open("demo.txt","r")

The open() function is used to open a file. It can open an existing file or create a new file depending on the selected mode.

📖 File Opening Modes

Mode Purpose
r Read data from file
w Write data into file (old data removed)
a Add new data at the end of file
x Create a new file

📚 Reading Data From File

file = open("demo.txt","r")

data = file.read()

print(data)

file.close()

The read() function reads complete content of a text file.

📄 Reading Line By Line

file = open("demo.txt","r")

line = file.readline()

print(line)

file.close()

The readline() function reads one line at a time.

✍ Writing Data Into File

file = open("demo.txt","w")

file.write("Welcome to Python Quest")

file.close()

The write mode creates a new file or replaces existing content.

➕ Appending New Data

file = open("demo.txt","a")

file.write("\nMaster Pyro")

file.close()

Append mode adds new content without deleting previous data.

🪄 Using with Statement

with open("demo.txt","r") as file:

    data = file.read()

    print(data)

The with statement automatically closes the file after use.

🐉 File Handling Adventure Rules

  • Always close files after completing operations.
  • Use meaningful file names.
  • Select correct file mode.
  • Handle errors using try-except.
  • Use encoding when working with different languages.

⚠ Common File Handling Errors

  • FileNotFoundError - File does not exist.
  • PermissionError - No permission to access file.
  • UnicodeDecodeError - Wrong encoding format.
  • Wrong File Mode - Incorrect operation selected.

🌟 Master Pyro's Final Tips

  • Use with open() whenever possible.
  • Keep backup copies of important files.
  • Use exception handling for safe programs.
  • Practice reading and writing files regularly.
