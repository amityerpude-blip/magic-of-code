📜 What is a Text File?

A text file is a collection of characters stored permanently on secondary storage. Python can create, read, modify and delete text files.

✨ Opening a File

file = open("demo.txt","r")

The open() function opens an existing file or creates a new one depending on the mode.

📖 Read Modes

Mode Description
r Read Only
w Write
a Append
x Create New File

📚 Reading Data

file = open("demo.txt","r")

print(file.read())

file.close()

✍ Writing Data

file = open("demo.txt","w")

file.write("Welcome to Python Quest")

file.close()

➕ Appending Data

file = open("demo.txt","a")

file.write("\nMaster Pyro")

file.close()

📄 Reading Line by Line

file = open("demo.txt")

print(file.readline())

file.close()

🪄 Using with Statement

with open("demo.txt","r") as file:

    print(file.read())

The file automatically closes after the block finishes.

⚠ Common Errors

  • FileNotFoundError
  • PermissionError
  • UnicodeDecodeError
  • Wrong File Mode

🏆 Best Practices

  • Always close files.
  • Use with open().
  • Use meaningful filenames.
  • Handle exceptions using try-except.
  • Keep backups of important files.
